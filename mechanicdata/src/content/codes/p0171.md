## Qu'est-ce que le code P0171 ?

Le **code défaut P0171** signale que le calculateur moteur (ECU) détecte un **mélange trop pauvre** sur le **banc 1** du moteur. "Trop pauvre" signifie qu'il y a trop d'air par rapport à la quantité de carburant dans la chambre de combustion.

Le rapport air-carburant idéal (stœchiométrique) est de **14,7:1** (14,7 parts d'air pour 1 part de carburant). Quand ce rapport dépasse cette valeur, le calculateur tente de compenser en injectant plus de carburant. Si la compensation atteint sa limite, le code P0171 apparaît.

> 💡 **Banc 1** : Sur un moteur V6 ou V8, c'est généralement le côté où se trouve le cylindre numéro 1.

## Symptômes du code P0171

Les symptômes varient selon la sévérité du problème :

- **Voyant moteur allumé** : Le témoin "Check Engine" s'illumine
- **Ralenti difficile** : Le moteur tourne de manière irrégulière au ralenti
- **Démarrage laborieux** : Le moteur met du temps à démarrer ou cale après démarrage
- **Hésitation** : Le moteur tousse ou hésite lors des accélérations
- **Perte de puissance** : Manque de reprise, surtout en charge
- **Consommation anormale** : Peut augmenter (le calculateur sur-compense)

## Causes principales du P0171

### 1. Entrée d'air parasite 🌬️
**La cause la plus fréquente.** De l'air s'infiltre dans le système après le débitmètre d'air, faussant les mesures.

**Points à vérifier** :
- Durites d'admission craquelées ou déconnectées
- Joint de collecteur d'admission défectueux
- Système de recyclage des gaz (PCV) défaillant
- Joints de corps papillon
- Système de ventilation du carter

**Diagnostic** :
- Inspection visuelle de toutes les durites
- Test au spray détecteur de fuite ou fumée
- Vérification du serrage des colliers

### 2. Débitmètre d'air (MAF) défectueux
Le capteur MAF mesure la quantité d'air entrant. S'il est sale ou défaillant, il envoie des données incorrectes.

**Diagnostic** :
- Nettoyer avec un spray spécial MAF
- Tester la tension de sortie (multimètre)
- Débrancher temporairement (si le problème disparaît = MAF coupable)

### 3. Pression de carburant insuffisante
La pompe à carburant, le filtre ou le régulateur ne fournissent pas assez de carburant.

**Diagnostic** :
- Test de pression de carburant (manomètre)
- Pression typique : 3-4 bars selon véhicule
- Écouter le bruit de la pompe au démarrage

### 4. Injecteurs bouchés
Les injecteurs encrassés ne pulvérisent pas correctement le carburant.

**Diagnostic** :
- Test de résistance des injecteurs
- Nettoyage aux ultrasons
- Test de pulvérisation (pattern test)

## Méthode de diagnostic étape par étape

### Étape 1 : Vérifier les codes associés
Si vous avez **P0171 + P0174**, le problème est généralement global (débitmètre, pression carburant).  
Si vous n'avez que P0171, concentrez-vous sur le banc 1.

### Étape 2 : Inspection visuelle
- Vérifiez toutes les durites d'admission
- Inspectez le filtre à air (très encrassé = flux d'air perturbé)
- Contrôlez le bouchon du réservoir (doit être étanche)

### Étape 3 : Nettoyer le MAF
Utilisez un spray nettoyant spécial pour débitmètre d'air (pas de WD-40 !).

### Étape 4 : Test de fumée
La méthode professionnelle : introduire de la fumée dans l'admission pour détecter les fuites.

### Étape 5 : Test pression carburant
Branchez un manomètre sur la rampe d'injection :
- Pression au contact : selon constructeur (généralement 3-4 bars)
- Pression doit rester stable

### Étape 6 : Lecture des données en temps réel
Avec une valise diagnostic, surveillez :
- **STFT** (Short Term Fuel Trim) : correction à court terme
- **LTFT** (Long Term Fuel Trim) : correction à long terme
- Valeurs positives élevées (+15% ou plus) = mélange pauvre confirmé

## Puis-je rouler avec un P0171 ?

**OUI à court terme**, mais avec précautions :

- ✅ Pas de danger immédiat majeur
- ⚠️ Consommation possiblement augmentée
- ⚠️ Performance réduite
- ❌ Ne pas ignorer longtemps (risque pour le catalyseur)

Il est recommandé de faire diagnostiquer le problème sous 1-2 semaines.

## Différence entre P0171 et P0174

- **P0171** : Mélange pauvre banc 1 (côté cylindre 1)
- **P0174** : Mélange pauvre banc 2 (côté opposé)

Si vous avez **les deux codes**, la cause est généralement commune aux deux bancs :
- Débitmètre d'air (MAF)
- Pression de carburant faible
- Fuite d'air globale (après la séparation des bancs)

## Codes défauts associés

- **P0174** : Système trop pauvre banc 2
- **P0172** : Système trop riche banc 1 (inverse)
- **P0175** : Système trop riche banc 2
- **P0420/P0430** : Efficacité catalyseur (conséquence du P0171)

## Astuces de pro 🔧

1. **Commencez par le plus simple** : filtre à air, nettoyage MAF
2. **Test au brake cleaner** : Pulvérisez autour des joints d'admission au ralenti. Si le régime change = fuite trouvée
3. **Freeze frame data** : Regardez les conditions d'apparition du code (ralenti ? accélération ?)
4. **Vérifiez le PCV** : Souvent oublié, mais fréquemment en cause

## Conclusion

Le **code P0171** peut avoir de nombreuses causes, mais un diagnostic méthodique permet de trouver rapidement le coupable. Commencez par les vérifications simples et gratuites (inspection visuelle, nettoyage MAF) avant d'envisager des réparations coûteuses. Dans 70% des cas, une fuite d'air ou un MAF sale est responsable.
